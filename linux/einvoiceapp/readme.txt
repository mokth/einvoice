1. install .Net 8 runtime
2. unzip the LinuxEInvoiceApp.zip in any folder.
3. go to the unzip folder, edit appsettings.json
    fill in the settings. Save it.
4. run the eInvoiceLinuxApp
    i. Load yuor Xml or Json file without signature.
    ii.  Click Generate Signature. All the value will show if not error.
    iii. Click Submit Document to submit to sandbox.
         Check the result show at the API result box.
    iv.  Click the Get Document / Get Document Detail to see the detail
5. Click Save to save the result to file for checking purpose.

Enjoy.
